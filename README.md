NutriCore 
Where Nutrition meets Fitness

NutriCore is a modern fitness & nutrition tracking web application built using Django.
It helps users track workouts, meals, calories, water intake, and overall fitness progress through a clean mobile-first interface.

---

🚀 Features

🏋️ Workout System

- Workout library
- Muscle group categorization
- Exercise detail pages
- Exercise images
- Workout progress tracking

🍎 Nutrition Tracking

- Food calorie library
- Macro tracking
- Meal logging
- Daily nutrition overview
- Water intake tracker

🥗 Recipe Library

- Healthy recipes
- Nutrition-based filtering
- Recipe categories
- Detailed recipe pages

📊 Progress Analytics

- Workout progress
- Monthly reports
- Fitness insights
- Progress visualization

⚙️ Admin System

- Custom admin dashboard
- Add/edit/delete foods
- Manage exercises
- Manage recipes
- User management

📱 Mobile Experience

- Fully responsive UI
- PWA (Progressive Web App)
- App-like experience on mobile
